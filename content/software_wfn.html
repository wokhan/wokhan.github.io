WFN (Windows Firewall Notifier)

Free / Open-source

The reknown open-source Windows Firewall extender!
Please note that WFN is maintained by the open-source community at GitHub (well, not all of it, we're only 3 😉).

How to get it

WFN is available on GitHub

Features

  • Connections listing
  • Real time connections mapping with routes
  • Bandwidth monitoring
  • Adapters informations
  • Windows Firewall status management
  • Notification popup for unknown outgoing connections (optional)
