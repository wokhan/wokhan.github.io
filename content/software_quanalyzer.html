QuAnalyzer

Free / Open-source

QuAnalyzer is a data-oriented tool I started working on around 2012. It has evolved a lot since, but has never been publicly released.

How to get it

While it has been open-sourced, no binaries have been released yet, as QuAnalyzer has been recently rewritten with WinUI3 and Uno Platform.
You can find more information on GitHub

Features

  • Data viewer with sorting / grouping abilities for virtually any data source
  • Data comparison from heterogenous sources (for instance a database versus a CSV or XLSX file)
  • Statistics and automatic pattern analysis
  • Monitoring and performance testing
  • ...
