Smart Tiles (aka Tiles for Outlook)

Free trial

A highly customizable calendar / agenda tiles creation app for Windows 10 / Windows 10 mobile.
Never miss an appointment again with this Windows calendar (Outlook on mobile) tile replacement!
You can also display all your upcoming friends birthdays on your start screen and stop forgetting about those.

How to get it

Smart Tiles is available in the Microsoft Store, with a free 14 days trial available to make your own opinion.

Features

  • Live tile
  • Various modes: calendar, agenda, birthdays...
  • Dedicated birthday screen
  • Highly customizable

Privacy policy

Depending on the feature the user would like to use, the Smart Tiles application can access the following personal data:
  • Windows Calendar entries (including events details)
    Data from the calendar is required when using the various events-based Smart Tiles options (after all, the app main goal is actually to display such information!)
  • Windows Contacts (solely their names and birthdays)
    Data from contacts is only used when birthday detection or listing is enabled.
  • Local files (when the user picks a custom background picture)
    For a bit of real customization, the user might choose to pick a custom picture ; of course it wouldn't work without actually accessing the selected files (and only those ones!)
Wokhan Solutions does not collect nor uses this information outside the application in any way.
